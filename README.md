# Royal Horizon Tours

A complete frontend-only tourism operating system demo for a premium Tanzanian tour operator.

## Stack

- HTML5
- Tailwind CSS
- daisyUI
- Vanilla JavaScript modules
- localStorage as the fake database
- No backend and no real payment/API calls

## Local Setup

```bash
npm install
npm run dev
```

Open the Vite URL shown in the terminal. The app stores demo bookings, payments, enquiries, reviews, newsletter leads, and travel forms in browser localStorage.

## Build

```bash
npm run build
```

The project is Vercel-ready as a static site. Vercel runs the build command and serves the root folder.
